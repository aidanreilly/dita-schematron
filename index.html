dita-schematron by jelovirt

ISO Schematron schema for DITA

ISO Schematron schema for DITA 1.1 and 1.2 that handles cases that DTDs or XML Schemas cannot control. The current implementation covers for example: * nesting links inside links or titles * deprecated attributes and attributes * attributes not intended for author use.
